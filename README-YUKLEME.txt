YÜKLEME TALİMATI

1) Bu zip dosyasını bilgisayarında aç.
2) İçindeki TÜM dosyaları GitHub Pages ana klasörüne yükle.
3) Eski HTML dosyalarının üzerine yazdır.
4) Mevcut görselleri silme: tamam.*, kaba.*, proje.*, kentsel.* dosyaları repoda kalmalı.
5) Yeni eklenen style.css, script.js ve bolgeler.html dosyalarını da mutlaka yükle.
6) Yükleme sonrası şu sayfaları kontrol et:
   - index.html
   - kentsel-donusum.html
   - projelerimiz.html
   - bolgeler.html
   - hizmetlerimiz.html

NOT: Bu paket görselleri silmez; sadece HTML/CSS/JS yapısını yeniler.


FARKLI KAPAK GÖRSELLERİ DÜZELTMESİ
----------------------------------
Bu pakette her iç sayfanın üst kapak görseli ayrı bağlandı.
style.css içinde page-hero artık --hero-bg değişkeniyle çalışıyor.

Örnek:
<section class="page-hero" style="--hero-bg:url('kaba.1.png')">

Mevcut görselleri silmeyin:
kentsel.1.png - kentsel.9.png
tamam.1.jpg - tamam.9.jpg
kaba.1.png - kaba.9.png
proje.1.png - proje.9.png

Yükleme:
Zip içindeki tüm dosyaları GitHub ana klasörüne yükleyip eski dosyaların üzerine yazdırın.
